## ‚ú® Main Features

- **Bypass lectures** - Mark all the video lectures as done.
- **Bypass discussions** - Mark all discussions as done.
- **Bypass readings** - Mark all the reading material as done.
- **Auto Quiz** - Automatically attempt quiz
- **AI Answers** - Solve all the quiz questions using AI
- **Custom API key** - Allows to use your own gemini API key

## ‚ÑπÔ∏è How to install?
1. Firstly, download and extract the extension zip from the latest release (or the above button)
2. Goto `chrome://extensions`, enable *Developer mode* from the top right.
3. Click on the *Load unpacked* button and select the extracted extension folder.
4. The extension is now installed. Now completing Coursera course is a cup of tea :)


> [!NOTE]
> The Auto Quiz/AI answers require you to enter Gemini API key. Get a key by clicking [here](https://aistudio.google.com/apikey)

## üßë‚Äçüíª Usage
When on the course page, where all the modules are shown in the left sidebar. Click 'Bypass course' and wait for the extension to complete the course. After that, reload the page.

Similarly, while on the quiz (opened) you can click 'Solve Quiz' button to automatically select the correct option. NOTE, this only works only and only on single choice questions (MCQ)


## üîë Getting API key
Getting a gemini API key is as simple as cooking a Maggi!
1. First of all, go to [this](https://aistudio.google.com/apikey) page.
2. Click on Create API key button
3. Select any project (or Generative language client)
> ![image](https://github.com/user-attachments/assets/8c2a918a-7d7e-4103-87e8-e3ffcc1b8516)

5. Copy the created API key
6. Paste the key in extention's UI key field and click save
